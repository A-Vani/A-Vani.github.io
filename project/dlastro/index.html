Deep Learning in Astronomy

Last updated on Mar 26, 2023
Photo by Astrobites

This was a self guided project. The aim was to understand the basics of deep learning techniques by using TensorFlow. Here a simple neural network is implemented to find whether a galaxy is spiral or elliptical based on photometric data or images obtained from SDSS. The data was provided by Ashish Mahabal during the 7BISS (2020) summer school.
